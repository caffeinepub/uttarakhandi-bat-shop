# Specification

## Summary
**Goal:** Create an e-commerce website for selling Uttarakhandi willow cricket bats at a fixed price of ₹600, with product catalog, shopping cart, and checkout functionality.

**Planned changes:**
- Build product catalog displaying Uttarakhandi willow cricket bats with images, descriptions, and ₹600 pricing
- Implement shopping cart allowing users to add bats, adjust quantities, and view totals
- Create checkout flow capturing customer name, delivery address, and phone number
- Store product data (name, description, image reference, price) in backend with query methods
- Store completed orders in backend with customer details, items, quantities, and total amount
- Apply traditional Indian marketplace visual theme with warm, earthy colors and elegant typography
- Display generated bat images from frontend/public/assets/generated directory

**User-visible outcome:** Users can browse Uttarakhandi willow cricket bats priced at ₹600, add items to a shopping cart, and complete checkout by providing delivery details, with a traditional Indian marketplace aesthetic.
